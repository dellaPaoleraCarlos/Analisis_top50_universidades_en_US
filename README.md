# Análisis de Universidades USA 2026 🎓
Proyecto de análisis de datos usando **Python**, **Pandas** y **Seaborn**.

## 📊 Visualizaciones
En este proyecto analizamos:
* Relación entre Ranking e Impacto de investigación.
* Distribución de instituciones públicas vs privadas.

### 🔬 Modelo Matemático
Para la regresión lineal calculamos el $R^2$ usando:
$$R^2 = 1 - \frac{SS_{res}}{SS_{tot}}$$

## 🚀 Cómo correr el script
1. Instalar dependencias: `pip install pandas seaborn matplotlib`
2. Correr: `python main.py`
